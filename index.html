üìã What is the Map Editor?

The Map Editor lets you create 50√ó50 tile-based maps for games. You can:

  • Import 16√ó16 pixel PNG tiles
  • Place tiles to build game levels
  • Import existing maps from TXT files
  • Save maps as PNG images or TXT files

üéØ Getting Started

1. Import Tiles: Click "Import Tiles" or drag & drop PNG files (16√ó16 pixels).

2. Select Tiles: Click on tiles in the palette to select them for painting.

3. Paint Map: Click and drag on the 50√ó50 grid to place tiles.

üìÅ File Requirements

Tiles: Must be 16√ó16 pixel PNG files. Name them with numbers (001.png, 002.png, etc.).
